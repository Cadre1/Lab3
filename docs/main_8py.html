Detailed Description

This program sets up the pins and timers required for a "MotorDriver" and an "Encoder" object and then sets up a "MotorControl" object.

It then waits for an input Kp value in which it will use to run a step response for the motor's position. The motor will have its PWM duty cycle set based off of the returned motor position from the encoder and passed through the proportional controller. The results of the step response are printed to the COM port to be read by "lab3.py"
